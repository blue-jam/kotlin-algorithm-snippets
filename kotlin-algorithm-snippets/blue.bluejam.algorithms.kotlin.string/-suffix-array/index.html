A suffix array (SA) constructed with O(n) times where n is the length of a string.

This implementation of SA contains an empty suffix. That is the first element of SA is always n. More precisely, this class adds \0 (null character) to the end of a string same as lots of articles add $ to a string. So, if you pass a string contains \0, this implementation might fail.
